Вопросы

Часто спрашивают

Почему хостинг бесплатный? Это надёжно?

GitHub Pages — официальный бесплатный сервис Microsoft, работает с 2008 года. Вы платите только за домен (~650 ₽/год). Сайт работает без абонентской платы бесконечно.

Что такое сайт с серверной логикой?

Это сайт с авторизацией, базой данных, личным кабинетом, API или приёмом платежей. В отличие от статики, там есть серверная часть, которая обрабатывает данные пользователей в реальном времени.

Смогу ли я сам редактировать сайт?

Да. Статический сайт — обычные HTML, CSS и JS. Тексты меняются в любом редакторе. Для сложных правок — обслуживание 500 ₽/мес.

Как выглядит оплата?

Онлайн через Robokassa: СБП, банковские карты, T-Pay, SberPay. Официальный чек сразу после оплаты.

Делаете ли вы интернет-магазины?

Да — базовый каталог с корзиной как дополнение к любому тарифу (+4 900 ₽). Полноценный магазин с серверной логикой — отдельный проект, обсудим.
